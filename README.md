# Search
***

This is a website, front-end for Google *Search*, Google Image *Search* and Google Advanced *Search*.  
It contains three pages:
* **Google Search** page
  * Links in the upper-right of the page to go to Image Search or Advanced Search.
  * The user can type in a query in the search bar, then click either `Google Search` button, and be taken to the Google search results for that page, or click on the `I'm Feeling Lucky` button and be taken directly to the first Google search result for the query, bypassing the normal results page.  
  
* **Google Image Search** page
  * Link in the upper right corner for the Google Search page
  * The user can type in a query in the search bar where clicking the `search button` will take the user to the *Google Image search results* for that page.
* **Google Advanced Search** page
  * Link in the upper right corner for the Google Search page 
  * The user can provide input for the following four fields :  
  
  ***Find pages with...***
  1.  *all these words:*                
  2.  *this exact word or phrase:*
  3.  *any of these words:*
  4.  *none of these words:*

Used for this website HTML, CSS. The ***CSS*** writen matches Google’s own aesthetics as best as possible using ***SASS***.
